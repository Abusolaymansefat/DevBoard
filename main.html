Question-2: What is the difference between innerHTML, innerText, and textContent ?
1.innerHTML: Gets or sets the HTML content inside an element, including all HTML tags. 2.innerText: Gets sets the visible text content of an element, excluding hidden elements (e.g., display: none or visibility: hidden). 3.textContent: Gets or sets the text content of an element, including all child nodes, regardless of visibility.

Question-4: What is event bubbling in the DOM?
একটি মেকানিজম, যেখানে কোনো এলিমেন্টে ঘটে যাওয়া ইভেন্ট (যেমন ক্লিক, মাউসওভার ইত্যাদি) তার প্যারেন্ট এলিমেন্টগুলিতে পর্যায়ক্রমে প্রপাগেট (propagate) বা ছড়িয়ে পড়ে। ইভেন্টটি টার্গেট এলিমেন্ট থেকে শুরু হয়ে DOM ট্রি এর উপরের দিকে (রুটের দিকে) চলতে থাকে।
